🌍 API Guinée Geo 🚀

Les données géographiques de la Guinée en accès libre 🇬🇳

Bienvenue sur l'API Guinée Geo, une API gratuite et ouverte recensant toutes les informations géographiques de la République de Guinée.

🔗 Routes rapides :

  • 📍 /regions : Afficher toutes les régions
  • 🏙️ /communes : Afficher toutes les communes
  • 📌 /quartiers : Afficher tous les quartiers
  • 🗺️ /prefectures : Afficher toutes les préfectures
  • 🛣️ /sous-prefectures : Afficher toutes les sous-préfectures

📘 Documentation de l’API

Toutes les routes renvoient des données au format JSON :

  • GET /regions – Liste des régions
  • GET /regions/search?name=Labé – Rechercher une région
  • GET /prefectures – Liste des préfectures
  • GET /prefectures/search?name=Kindia – Rechercher une préfecture
  • GET /communes – Liste des communes
  • GET /communes/search?name=Matoto – Rechercher une commune
  • GET /sous-prefectures – Liste des sous-préfectures
  • GET /sous-prefectures/search?name=Koulé – Rechercher une sous-préfecture
  • GET /quartiers – Liste des quartiers
  • GET /quartiers/search?name=Almamya – Rechercher un quartier

💡 Tous les résultats sont renvoyés au format JSON. Les recherches par nom retournent un objet ou une erreur 404.

ℹ️ À propos

Cette API recense les subdivisions administratives officielles de la Guinée : régions, préfectures, sous-préfectures, communes et quartiers. Elle peut être utilisée pour des applications cartographiques, des projets open data, ou des analyses territoriales.
