🌍 API Guinée Geo 🚀

Bienvenue sur l'API Guinée Geo, une API gratuite et ouverte recensant toutes les informations géographiques de la Guinée.

🔗 Routes disponibles :

  • 📍 /regions : Afficher toutes les régions
  • 🏙️ /communes : Afficher toutes les communes
  • 📌 /quartiers : Afficher tous les quartiers
  • 🗺️ /prefectures : Afficher toutes les préfectures
  • 🛣️ /sous-prefectures : Afficher toutes les sous-préfectures
